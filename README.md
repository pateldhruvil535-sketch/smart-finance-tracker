# 💸 Smart Finance Tracker

Full Stack Smart Finance Tracker with AI insights, budget management, and multi-currency support.

## 🚀 Features
- Track Income & Expenses
- Budget Management
- AI Insights & Tips
- Charts & Analytics
- Multi-Currency Support
- Responsive UI

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
